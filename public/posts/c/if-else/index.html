if…else statements

2023-01-07 — Written by Andy Sukowski-Bang

An if statement can be followed by an optional else statement, which executes when the Boolean expression is false.

Example program # 

// necessary for `bool`
#include <stdbool.h>

int main (int argc, char *argv[])
{
	// could also be `a`, `5`, `%`
	char some_character = 'R';

	// positive or negative
	int my_integer = -7642;

	// also positive or negative
	float pi = 3.1415926;

	// useful for conditions (later in tutorial)
	bool going_to_school = true;

	return 0;
}
