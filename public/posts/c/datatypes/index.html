Datatypes

2023-01-05 — Written by Andy Sukowski-Bang

Data types in C refer to an extensive system used for declaring variables or functions of different types. The type of a variable determines how much space it occupies in storage and how the bit pattern stored is interpreted.

These are the most basic datatypes in C:

Type Explanation
char character (e.g. 'a' or '1')
int signed integer from INT_MIN to INT_MAX
float floating-point number like 3.141
bool either true or false

Example program #

The following program stores variables of different datatypes in memory

// necessary for `bool`
#include <stdbool.h>

int main (int argc, char *argv[])
{
	// could also be `a`, `5`, `%`
	char some_character = 'R';

	// positive or negative
	int my_integer = -7642;

	// also positive or negative
	float pi = 3.1415926;

	// useful for conditions (later in tutorial)
	bool going_to_school = true;

	return 0;
}
